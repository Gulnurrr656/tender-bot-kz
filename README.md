# Tender Bot (Kazakhstan Goszakup)

Telegram-бот для мониторинга государственных закупок РК.

## Возможности
- Поиск и фильтрация лотов
- Авто-уведомления в Telegram
- Фильтр по статусу и ключевым словам
- Защита от дублей и Flood control
